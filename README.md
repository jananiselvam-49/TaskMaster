# TaskMaster

## Description
A simple and responsive To-Do app built using HTML, CSS, and JavaScript that allows users to add, complete, and delete tasks with date and time support for better productivity.
Features include:
- Add tasks with date & time
- Mark tasks as completed
- Delete tasks
- Motivational quotes on task completion

## Features
- Overlay popups for adding tasks
- Dynamic task cards
- Completed task counter
- Motivational alerts
